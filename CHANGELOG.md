# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-27

### Added
- Initial release
- Vue 3 + Vite frontend application
- Tailwind CSS untuk styling
- Spotify URL input dan validation
- Real-time download progress display
- Responsive design (mobile, tablet, desktop)
- Axios integration untuk API calls
- Vue Router untuk navigation
- Docker support dengan Nginx
- Complete documentation (README, CONTRIBUTING, SECURITY)

### Features
- Clean dan modern UI
- Download tracking dengan progress bar
- Error handling dan user feedback
- Loading states untuk async operations
- Mobile-friendly responsive design
